# 変数名や関数名を、どの { ... } の中で定義するか
* 次の２点のことを考慮する

(1) 名前をどの範囲で有効にさせたいか

(2) その変数や関数は、どこに所属するのが自然であるか

# おまけの知識
```
void func_A()
{
	...
	
	void func_B()
	{
		...
	}

	...
}
```

* 上のようになっている場合、`func_B` は `func_A` の中からしか呼び出すことができない。そのため、`func_B` は __「ローカル関数」__ と呼ばれる。

* 変数名や関数名が見える範囲のことを **「スコープ」** という。上の例で言うと、「func_B のスコープは、func_A の中だけになる」となる。
