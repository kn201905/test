# Ｎ進数の解説ありがとう^^
* プログラムを書くときに利用するのが、２進数と１６進数だから、その結びつきが分かるようにしてもらえるとありがたいです。

* Ｎ進数とは、Ｎ個の記号を用いて個数を表現する手法と考えると分かりやすいかも？  
３進数だったら 0, 1, 2 の３つの記号を用いて個数を表すという感じ。個数が３つになったら桁上りをする。  
４進数だったら 0, 1, 2, 3 の４つの記号を用いて個数を表して、個数が４つになったら桁上りをする。
